ICT Syllabus

Grade 10 Syllabus

Unit 1: தகவல் மற்றும் தொடர்பாடல் தொழினுட்பவியல்

Introduction to ICT concepts and communication technologies

Unit 2: கணினியை இனங்காண்போம்

Understanding computer systems and components

Unit 3: கணினியில் தரவுகள் எடுத்துக்காட்டப்படும் முறைகள்

Data representation methods in computers

Unit 4: தருக்க வாயில்களும் பூலியன் தர்க்கங்களும்

Logic gates and Boolean algebra

Unit 5: பணிசெயல் முறைமைகள்

Operating systems fundamentals

Unit 6: சொல் முறைவழிப்படுத்தல்

Word processing techniques

Unit 7: இலத்திரனியல் விரிதாள்கள்

Spreadsheet applications

Unit 8: இலத்திரனியல் நிகழ்த்துகை

Electronic presentations

Unit 9: தரவுத்தளம்

Database management systems

Grade 11 Syllabus

Unit 1: பிரச்சினைகளைத் தீர்ப்பதற்குச் செய்நிரல்களை எழுதுதல்

Problem solving and programming

Unit 2: தகவல் முறைமை விருத்திச் செயன்முறை

Information system development process

Unit 3: இணையம் மற்றும் மின்னஞ்சல்

Internet and email technologies

Unit 4: பல்லூடகப் பயன்பாடு

Multimedia applications

Unit 5: எளிய இணையத்தளங்களை வடிவமைத்தல்

Basic web design principles

Unit 6: சமூகமும் தகவல் மற்றும் தொடர்பாடல் தொழினுட்பவியலும்

Social impacts of ICT
